Project Overview

Developed a cross-sport athlete success prediction model using data from NFL, NBA, and FIFA. The system uses advanced machine learning techniques to predict 10-year career trajectories based on physical attributes and position-specific features, helping young athletes understand their potential career paths.

Technical Approach

  • Collected and preprocessed data from 1,800+ professional athletes across three major sports leagues
  • Implemented hierarchical position encoding to reduce dimensionality while preserving position-specific information
  • Optimized 5 regression models (Linear, Ridge, Lasso, Random Forest, Gradient Boosting) using GridSearchCV
  • Achieved sport-specific best performance with cross-validation and hyperparameter tuning
  • Deployed interactive web application for real-time predictions and visualizations
